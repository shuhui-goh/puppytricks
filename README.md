# Puppy Tricks

A gesture-controlled corgi that reacts to your hand movements via webcam.

Built with MediaPipe Gesture Recognizer and vanilla JS. No frameworks, no backend.

## Gestures
- ✋ Open palm → Stay
- ☝️ Point up → Sit
- 👍 Thumbs up → Good boy!

## Run locally
```
python3 -m http.server 8080
```
